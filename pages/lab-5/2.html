Схема URL

JavaScript можна використовувати для динамічного створення URL-адрес. Це можна зробити за допомогою функції window.location.href. 
            
                var url = window.location.href;
                url += "page.html";

Обробник подій

JavaScript можна використовувати для обробки подій, таких як клік, наведення курсора миші або завантаження веб-сторінки. Це можна зробити за допомогою обробників подій, які прив'язуються до елементів DOM. 
            
                var button = document.getElementById("submit");
                button.addEventListener("click", function() {
                // Do something
                });

Підстановка

JavaScript можна використовувати для підстановки об'єктів в HTML-код. Це можна зробити за допомогою функції document.querySelector(). 
            
                var person = {
                    name: "John Doe",
                    age: 30
                  };
                  
                  var p = document.querySelector("p");
                  p.innerHTML = person.name + " is " + person.age + " years old";

Вставка (тег script)

JavaScript можна вставляти в HTML-код за допомогою тега script. Тег script містить код JavaScript, який буде виконаний браузером. 
            
                <script>
                    //JavaScript code
                </script>
